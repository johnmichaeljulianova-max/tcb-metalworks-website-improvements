# TCB Metal Works — New Website (Work in Progress)

A ground-up redesign of the TCB Metal Works website, built to present the company as a
premium custom metal fabrication shop and to grow search traffic — with a special focus on
the **zoo & animal enclosure** niche alongside the core fabrication services.

enclosures section, and the articles/blog.

## What's in this repository

| Folder | What it is |
| --- | --- |
| [`tcb-website/`](tcb-website/) | The complete new website — hand-built HTML, CSS, and JavaScript (no page builder, loads fast on any device). |
| [`_scraper/`](_scraper/) | The behind-the-scenes build & quality-check scripts used to generate and validate the pages. |
| `TCB-Website-Progress-Report.pdf` | A written summary of the work completed so far. |

## Highlights so far

- **20 pages** built from scratch with a consistent, premium "elevated industrial" design
- A dedicated **Zoo Enclosures** section positioned as the flagship service
- **13 in-depth articles & blog posts** (~30,000 words) covering zoo enclosure design,
  materials, and fabrication — built to attract search traffic
- Fully **mobile-responsive**, accessible, and search-engine optimized (titles, descriptions,
  structured data, sitemap, internal linking)
- Subtle, on-brand touches — animated welding sparks in the hero, a "machined metal" sheen
  on section backgrounds

## Running it locally (optional, for developers)

The site is plain static files — no build step required.

```bash
cd tcb-website
node server.mjs      # then open http://localhost:5180
```
